### Controls:
- **A/D**: Move Character
- **Space**: Jump
- **Left Click**: Attack
- **Right Click**: Dash Attack
- **Esc**: Pause (or leave the credits)

### How to play through this repo:
1. Clone/download this repo
2. In your IDE, set res as **resource folder** and set src as **source folder**
3. Play and enjoy ig
